# Rowify — WhatsApp Data Parser
## A gentle tool to tidy WhatsApp lists for easy human review.

### 🧩 Overview
Rowify is a Streamlit‑based app that converts messy WhatsApp message exports into structured, review‑ready spreadsheets.
It’s designed for real‑world use — where data is imperfect, human context matters, and reliability beats complexity.

### 💡 Core Idea
A data solution doesn’t have to eliminate humans from the loop —
it just has to make humans dramatically more effective.
Rowify doesn’t aim for perfection.
It aims for trustworthy automation — the kind that saves hours of manual work while keeping humans in control.

### ⚙️ Features
- Parses WhatsApp .txt exports into clean, structured rows
- Handles multi‑person messages, mixed tokens, and real‑world irregularities
- Adds a Needs Review column for human supervision
- Exports to Excel with color‑coded review flags
- Includes a Reviewer Notes sheet for contextual guidance

### 🧠 Philosophy
In the real world, a tool that reliably reduces manual work
is often better than a technically elegant system that breaks in edge cases or never reaches users.
Rowify is built for that world — where simplicity, clarity, and human judgment matter most.

### 🚀 Usage
- Export your WhatsApp chat as .txt
- Upload it to Rowify
- Review the parsed table

Download the Excel file with review flags

Make quick, confident decisions
